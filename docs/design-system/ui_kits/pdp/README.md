# KAL Global — PDP UI Kit

Pixel-fidelity recreation of the **Neelibhringadi Keram** Product Detail Page,
extracted from `neeli-prototype/index.html` and the provided Figma.

- `index.html` — interactive click-through of the full PDP.
- Shares `../../colors_and_type.css` and `../../assets/`.

The original prototype is 980 lines of hand-authored HTML/CSS/JS. This kit
mirrors its architecture: sticky header, gallery + thumb strip, stats bar,
product info + ritual panel + delivery badges + ATC, trust markers, cross-sell,
mechanism, honest sequence, timeline, ingredients grid, triple-milk, what's-in/
what's-out, testimonials, application guide, compare table, accordion, reviews,
closing echo, sticky buy bar.

## Components covered

- **Header** (frosted glass, logo, nav, cart)
- **Gallery** (crossfade slides, thumb tray inside stage, arrows)
- **Stats bar** (4-cell credential row)
- **Product info** (badge, H1, italic subtitle, meta, ritual guide, delivery badges)
- **Trust markers** (5-cell: 1945 / GMP / ISO / FDA / AYUSH)
- **Cross-sell strip** (rust band, amber CTA)
- **Mechanism** (brown eyebrow, serif H2, centered body)
- **Honest sequence** (dark green, centered quote)
- **Timeline** (3-step with amber-bordered thumbnails)
- **Ingredient cards** (4 herbs, data-attribute CSS background images)
- **Milk cards** (3 milks, circle icons)
- **What's in / What's out** (divider column, pale green/red circles)
- **Testimonials** (dark green, serif quote, result tag)
- **Application guide** (3 steps: warm · apply · massage)
- **Compare** (Us vs generic)
- **Accordion** (4 items, chevron rotate)
- **Reviews** (aggregate + cards + outline CTA)
- **Closing echo** (dark green, serif text — no CTA button)
- **Sticky buy bar** (fixed bottom)
